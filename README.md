# 💻 Prácticas de HTML, CSS y JavaScript

Repositorio con ejercicios prácticos de desarrollo web.
Cada carpeta es un ejercicio independiente.

---

## ⚙️ Cómo usar (en cualquier PC)

1. Abrir la terminal (CMD)

2. Ir a la carpeta donde querés clonar (ej: Escritorio):

   ```
   cd Desktop
   ```

   👉 El repositorio se va a descargar en esta ubicación

3. Clonar el repositorio:

   ```
   git clone URL_DEL_REPO
   ```

4. Entrar al repo:

   ```
   cd NOMBRE_DEL_REPO
   ```

5. Ir al ejercicio que quieras:

   ```
   cd 01-HTML
   ```

6. Abrir:

   * Doble click en `index.html`

---

## ⚠️ Nota

Git no permite clonar una sola carpeta, pero cada ejercicio puede usarse de forma independiente entrando a su directorio.
